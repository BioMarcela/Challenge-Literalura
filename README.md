## Challenge-Literalura

Literalura es una aplicación de consola desarrollada en Java con Spring Boot que permite buscar, listar y almacenar libros y autores utilizando datos obtenidos de una API externa. El proyecto integra Spring Data JPA para la persistencia en una base de datos PostgreSQL.

## Características

- Buscar libros por título y mostrar información detallada (título, idiomas, descargas, autores).
- Listar todos los libros registrados en la base de datos.
- Listar todos los autores registrados.
- Consultar autores que estuvieron vivos en un determinado año.
- Listar libros por idioma.
- Almacenar automáticamente libros y autores en la base de datos si no existen.
- Evitar duplicados en la base de datos mediante validaciones en el código y restricciones.

## Tecnologías

- Java 17
- Spring Boot 3.5.11
- Spring Data JPA
- PostgreSQL
- Hibernate ORM 6.6
- Maven como gestor de dependencias
